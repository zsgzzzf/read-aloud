微软“朗读” TTS 转发器

运行于 Cloudflare Workers 上的微软“大声朗读”转发器。

直接使用 GitHub

“大声朗读” 是什么？

“大声朗读” (Read aloud) 是微软为一系列自家服务提供的文字转语音服务。由于大家基本上都使用的是 Edge 浏览器上的接口，所以也被时常称为 "Edge TTS "。

自行部署

请前往 GitHub 项目页面查看。

导入

目前支持阅读、爱阅记、源阅读的导入。
